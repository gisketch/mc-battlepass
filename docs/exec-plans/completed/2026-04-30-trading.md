# Trading

## Goal

Add player-to-player trading with item slots, chowcoin offers, request/accept by right-click, visual glow cues, and solo debug testing support.

## Acceptance Criteria

- Right-clicking another player creates a trade request; reverse right-click accepts while pending.
- Pending request glows the target for requester and requester for target, then clears on accept, decline, expiry, logout, or distance/world invalidation.
- Accepted trade opens a UI for both players with 27 own offer slots, 27 other offer slots, own inventory access, other inventory preview dimmed/locked, chowcoin offer controls, ready/confirm flow, and cancel/return behavior.
- Trade commits atomically: items and chowcoins move only when both players confirm valid offers.
- Solo dev test path exists without needing another human.
- Build passes.
