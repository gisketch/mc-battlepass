# Mission Curation

## Goal

Create a repo-local curation document for CKDM mission and quest expansion that separates implemented hooks, code-needed hooks, and configuration reward bands.

## Acceptance Criteria

- Current usable mission hooks are listed.
- Proposed uncreated hooks are grouped by code ownership.
- NPC, weekly, and permanent reward bands are tied to early/mid/late income targets.
- Follow-up code work and configuration work are clear.
