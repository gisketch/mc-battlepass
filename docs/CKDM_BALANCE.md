# CKDM Balance

This pass makes CKDM conservative by default: slow 1000-level seasons, visible Cozy/Combat relic tokens, limited mission XP, and curated stores preserved for targeted price review.

## Generated Configs

Run:

```powershell
.\scripts\generate-ckdm-balance.ps1
```

The generator writes:

- `runs/client/config/gisketchs_chowkingdom_mod/battlepass/passes/cozy.toml`
- `runs/client/config/gisketchs_chowkingdom_mod/battlepass/passes/combat.toml`
- `runs/client/config/gisketchs_chowkingdom_mod/relic_roulette/pools/common_cozy_relics.toml`
- `runs/client/config/gisketchs_chowkingdom_mod/relic_roulette/pools/rare_cozy_relics.toml`
- `runs/client/config/gisketchs_chowkingdom_mod/relic_roulette/pools/common_combat_relics.toml`
- `runs/client/config/gisketchs_chowkingdom_mod/relic_roulette/pools/rare_combat_relics.toml`
- `docs/generated/ckdm-balance-summary.md`

Store configs are intentionally not generated by this script. They are already curated and should only receive targeted price edits after review.

Do not hand-maintain 1000 battlepass reward tiers. Update the generator tables, regenerate, then review the summary.

## Battlepass Shape

- Level formula remains `XP / 100`.
- Max season tier is level 1000 at `100000 XP`.
- Both Cozy and Combat passes have one reward every level.
- Cozy level 500 grants an Elytra in addition to the rare Cozy relic token.
- Every 5 levels gives a better utility reward.
- Every 10 levels or special milestone gives a prominent reward.
- No raw direct XP loops are generated.

## Relic Token Placement

Safe Slow launch placement:

- Cozy common: levels 50, 200, 300, 750
- Cozy rare: levels 100, 500, 1000
- Combat common: levels 50, 200, 300, 750
- Combat rare: levels 100, 500, 1000

Total guaranteed battlepass relic tokens: 14.

## Elytra Gate

- Elytra is not removed from End loot, item frames, or other loot sources.
- All Elytra are unwearable until overall Battlepass Level 500.
- The level gate blocks right-click equip, use start, chest-slot equipment changes, and periodic equipped-state checks.
- Locked Elytra show the same inventory lock overlay style used by class item locks.

## Tech Mod Gates

- Tech mods are locked by namespace interaction until the player earns that mod's license.
- Default server-wide shipping thresholds: Create/Bulma at 100,000, Ars Nouveau/Howl at 150,000, and Oritech/Rick Sanchez at 200,000 shipped chowcoins.
- Default personal license fee is 10,000 chowcoins each, after NPC proof steps.
- Gates block right-click item use, right-clicking tech blocks, block placement, use-start, equipment changes, and equipped-state checks. Breaking blocks remains allowed.
- Default gated namespaces live in `config/gisketchs_chowkingdom_mod/tech_licenses/licenses.toml`; always-banned item/block patterns can hard-ban specific outliers such as abusive Ars automation.
- Tech expert daily missions unlock only after the matching player license and do not consume the normal NPC daily quest cap.

## XP Bands

- Weekly normal missions: 150-300 XP target.
- Weekly hard missions: 350-500 XP target.
- Weekly rare/special missions: 600-900 XP only when limited.
- Permanent early milestones: 150-400 XP.
- Permanent mid milestones: 500-1000 XP.
- Permanent late milestones: 1200-2500 XP.

The generated configs currently use 220-400 XP normal/hard weeklies, 600 XP special weeklies, and permanent chain rewards capped at 2000 XP.

## Excluded From Normal Pass Rewards

- Nether Star
- Netherite Ingot
- Totem of Undying
- Enchanted Golden Apple
- Shulker Shell
- Random Enchanted Book
- Old generic relic token rewards

Old `common_relic_token` and `rare_relic_token` remain registered for compatibility, but new pass rewards use the four Cozy/Combat token pools.

## Store Bands

- Tiny convenience: 50-150
- Common seeds/supplies: 100-600
- Useful convenience: 1500-4000
- Rare utility: 5000-12000
- Weekly rare collectible: 12000-30000
- Prestige cosmetic/set: 50000+

Stores should not sell direct relic tokens, Master Balls, raw boss rewards, netherite progression, or economy-breaking tech shortcuts.
